Spice Console Setup

To use an encrypted spice session, please follow the following steps:

Prerequisites

Make sure your local .spicec folder exists: 

mkdir -p ~/.spicec
Add this Cluster's CA Certificate to your trust store: 
cat << EOF > ~/.spicec/spice_truststore.pem
{{ spiceCa }}
EOF

Linux

Install the remote-viewer application: 

apt-get install virt-viewer

Mac OS X

Install the remote-viewer application from this URL.

You can not use the "Spice Console"-Button on OS X as of now. To open a console (do this from your home directory, RemoteViewer uses relative paths which is a bug.): 

/Applications/RemoteViewer.app/Contents/MacOS/RemoteViewer --spice-ca-file="~/.spicec/spice_truststore.pem" --spice-host-subject="CN=ganeti.example.com" spice://{{ instance.pnode }}?tls-port={{ instance.network_port }}
